# personnel_registration_delete_leave-system



# Bu projedeki temel amacım şirketlerin işçi-personel kontrolünü kolaylaştıran bir sistem tasarlamaktır. Python class ve fonksiyon metotlarını kullandığım bu proje birçok özellik barındırmaktadır. Öncelikle diğer projelerimde de kullandığım güvenlik sistemini burada da kullandım. Usrname ve password doğru girilmediği taktirde menüye giriş yapılamayacaktır. Kulllanıcı username ve password u doğru girdiğinde 7 bölümden oluşan bir menü ile karşılaşır. Bu menüdeki bölümleri ve amaçlarını anlatacak olursam: 1- sisteme yeni işçi eklenmesini sağlar. Bu bölümde kullanıcıdan inputlar alınarak isim, iletişim adresi, çalıştığı alan, maaş, kaç gün izin hakkı olduğu gibi bilgiler sisteme girilir ve bir txt dosyasına yazdırılır. 2- Bu seçeneğin amacı işçiye izin verebilmektir. İşçiye en başta belirli bir izin alma hakkı tanımlanır. Eğer ki verilecek izin bu haktan daha az ise işlem gerçekleşir ve ekrana hangi ID li işçiye kaç gün izin verildiği ve kaç gün daha hakkı kaldığı bilgisi yazdırılır. 3-Bu seçeneğin amacı ID si girilen işçinin tüm bilgilerini görmektir. 4-Personelin kaydının silinmesini sağlar. 5- Personelin bilgilerinin güncellenmesini sağlar. 6-Personelin izin hakkı sayısının güncellenmesini sağlar. Bütün bu işlemler personelin ID si ile yapılmaktadır. 7-Sistemden çıkış yapılıp tekrardan şifre ekranına döndürülmeyi sağlar.



# My main goal in this project is to design a system that facilitates the worker-personnel control of companies. This project, in which I use Python class and function methods, has many features. First of all, I used the security system that I used in my other projects as well. If the usrname and password are not entered correctly, it will not be possible to enter the menu. When the user enters the username and password correctly, he/she will see a menu consisting of 7 sections. If I were to explain the sections in this menu and their purposes: 1- It allows new workers to be added to the system. In this section, inputs are taken from the user and information such as name, contact address, area of employment, salary, and how many days leave are entered into the system and printed in a txt file. 2- The purpose of this option is to allow the worker to leave. The employee is initially given the right to take a certain leave. If the leave to be granted is less than this right, the transaction takes place and the information on how many days off is given to the employee with which ID and how many more days are left is printed on the screen. 3- The purpose of this option is to see all the information of the worker whose ID is entered. 4-It allows the deletion of the personnel's record. 5- Provides updating of personnel information. 6-Allows the personnel to update the number of leave rights. All these operations are done with the ID of the personnel. 7- It enables to exit the system and return to the password screen again.



# Mein Hauptziel in diesem Projekt ist es, ein System zu entwerfen, das die Arbeiter-Personal-Steuerung von Unternehmen erleichtert. Dieses Projekt, in dem ich Python-Klassen- und -Funktionsmethoden verwende, hat viele Funktionen. Zunächst habe ich das Sicherheitssystem verwendet, das ich auch in meinen anderen Projekten verwendet habe. Wenn der Benutzername und das Passwort nicht korrekt eingegeben werden, kann das Menü nicht aufgerufen werden. Wenn der Benutzer den Benutzernamen und das Passwort richtig eingibt, sieht er ein Menü, das aus 7 Abschnitten besteht. Wenn ich die Abschnitte in diesem Menü und ihre Zwecke erklären müsste: 1- Es ermöglicht das Hinzufügen neuer Mitarbeiter zum System. In diesem Abschnitt werden Eingaben des Benutzers übernommen und Informationen wie Name, Kontaktadresse, Einsatzbereich, Gehalt und wie viele freie Tage in das System eingegeben und in eine txt-Datei gedruckt. 2- Der Zweck dieser Option besteht darin, dem Arbeitnehmer das Verlassen zu ermöglichen. Der Arbeitnehmer erhält zunächst das Recht, einen bestimmten Urlaub zu nehmen. Ist der zu gewährende Urlaub geringer als dieser Anspruch, erfolgt die Transaktion und die Information, wie viele freie Tage der Mitarbeiter mit welchem Ausweis erhält und wie viele Tage noch übrig sind, wird auf dem Bildschirm ausgedruckt. 3- Der Zweck dieser Option besteht darin, alle Informationen des Arbeitnehmers anzuzeigen, dessen ID eingegeben wurde. 4-Es ermöglicht die Löschung des Personaldatensatzes. 5- Stellt die Aktualisierung von Personalinformationen bereit. 6-Ermöglicht dem Personal, die Anzahl der Urlaubsrechte zu aktualisieren. Alle diese Operationen werden mit der ID des Personals durchgeführt. 7- Ermöglicht das Verlassen des Systems und die Rückkehr zum Passwortbildschirm.
